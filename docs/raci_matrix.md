# RACI Matrix: MS Teams Dashboard Integration

## 1. Project Roles
- **Project Manager (PM)**: Overall project leadership, risk management, stakeholder communication.
- **Product Owner (PO)**: Backlog grooming, feature prioritization, acceptance criteria.
- **Team Lead (TL)**: Technical oversight, architecture, and team management.
- **Backend Developer (BE)**: MS Graph API integration, data processing logic, and DB management.
- **Frontend Developer (FE)**: Dashboard UI, visualization components, and user interactions.
- **UI/UX Designer**: Mockups, design guidelines, and user experience.
- **IT Admin**: Microsoft 365 environment setup and permission management.

---

## 2. RACI Definitions
- **R**: Responsible (Performs the work)
- **A**: Accountable (The one who ultimately answers for the result)
- **C**: Consulted (Provides input/feedback)
- **I**: Informed (Needs to be kept updated)
